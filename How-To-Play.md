

Overview


In this document, you will learn how to play Bash and/or set it up for yourself to be played locally/over LAN! Bash includes two methods of playing, one, through a pre shipped Itch.io page that can be played in the browser with 0 setup required. If you would like to go more in depth and have control, while also able to look at the codebase, there are instructions on cloning the repo and installing all dependecies, and getting started.

Play On Itch

Bash is published to an itch io page. In this, you will have access to the full game 
https://teomendoza.itch.io/bash
